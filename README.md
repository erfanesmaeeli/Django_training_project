پروژه آموزشی جنگو

-------------

ابتدا یک محیط مجازی با دستورزیر ایجاد کنید:
virtualenv .env

سپس آنرا فعال کنید:

ویندوز:
.env\Scripts\activate

لینوکس:
.env/bin/activate

سپس پکیج‌های مورد نیاز را نصب کنید:
pip install -r requirements.txt

با دستور زیر پروژه را ران کنید:
python manage.py runserver
