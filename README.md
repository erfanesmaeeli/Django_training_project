<b>پروژه آموزشی جنگو
</b>

<hr>

ابتدا یک محیط مجازی با دستور زیر ایجاد کنید

<br>

<code>
virtualenv .env
</code>


<br>
سپس آنرا فعال کنید

<br>

ویندوز:

<br>

<code>
.env\Scripts\activate
</code>

<br>

لینوکس:

<br>

<code>
.env/bin/activate
</code>

<br>

سپس پکیج‌های مورد نیاز را نصب کنید

<br>

<code>
pip install -r requirements.txt
</code>

<br>

با دستور زیر پروژه را ران کنید

<br>

<code>
python manage.py runserver
</code>
